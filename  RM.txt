Написать чат используя сокеты.

Цель проекта:
1. Изучить работу с сокетами
2. Изучить основные техники создания протоколов обмена данными
3. Изучить библиотеку readline для создания command line interfaces (CLI)
4. Изучить основные техники создания CLI

Требования к серверу
	1. Использовать сокеты с протоколом TCP для обмена данными+
	2. Поддержка нескольких клиентов (соединений) +
	3. Поддержка нескольких "комнат" для общения+
	4. Поддержка личных сообщений (PM)+
	5. Хранение данных о пользователях и сообщений в БД +
	6. Рассылка уведомлений клиентам о новых PM или упоминаний +

Требования к клиенту
	1. Установка соединения с сервером с помощью логина и пароля +
	2. Интерфейс для переключения комнат, просмотра и отправки личных сообщений
	3. Возможность получения уведомлений от сервера, когда приходит PM или упоминание в чате
	4. Возможность выделять текст цветами+
	5. Регистрация нового клиента +
	6. Поддержка autocomplete для команд и имен пользователей+
